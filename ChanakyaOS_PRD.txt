# **1. PROJECT OVERVIEW ****&**** GOALS**

## **1.1 Vision Statement**

ChanakyaOS is an AI-powered career operating system designed to help students and young professionals build the right profile, roadmap, and proof of readiness required to secure internships, jobs, and long-term career success. Named after Chanakya — India's ancient master of strategy and wisdom — the platform embodies the principle that the smartest, most prepared person wins the career game, not just the luckiest.

## **1.2 Core Problem Statement**

The biggest problem in today's internship and job market is NOT the lack of opportunities. The real problem is that most students are simply not career-ready in a structured and optimized way.

### **Students Consistently Suffer From:**

- No clear career path — following random YouTube advice instead of structured guidance

- Incomplete or weak resumes — failing ATS systems and recruiter scans

- Unoptimized LinkedIn profiles — invisible to recruiters

- Irrelevant certifications — spending money on credentials that don't matter

- No project portfolio — lacking proof-of-work that employers demand

- No roadmap — learning skills without direction or sequence

- Background-career misalignment — e.g. Commerce student competing blindly with CS engineers

## **1.3 The Solution**

ChanakyaOS becomes a complete, one-stop career-building ecosystem that analyzes the student's current position, identifies every gap in their profile, generates multiple realistic career pathways tailored to their background, and guides them step-by-step through becoming genuinely employable — not just qualified on paper.

# **8. RECOMMENDED TECHNOLOGY STACK**

| **Layer** | **Recommended Stack** | **Rationale** |
| --- | --- | --- |
| **Frontend** | Next.js 14 (React) + Tailwind CSS + Framer Motion | SSR for SEO-critical pages, fast performance, component reuse across dashboard |
| **Backend** | Node.js + Express or Python FastAPI | FastAPI preferred for AI/ML integration; Node.js for real-time features |
| **Database** | PostgreSQL (primary) + Redis (cache/sessions) + Pinecone (vector DB) | Relational for user data, Redis for performance, Pinecone for AI semantic search |
| **AI/LLM** | OpenAI GPT-4 Turbo / Anthropic Claude 3.5 (via API) | GPT-4 for complex reasoning; Claude for long-context profile analysis |
| **Resume Parsing** | AWS Textract + custom NLP pipeline (SpaCy) | High accuracy on varied resume formats including Indian resume styles |
| **Authentication** | NextAuth.js / Auth0 with Google + LinkedIn SSO | Industry standard, handles OAuth for LinkedIn import seamlessly |
| **File Storage** | AWS S3 or Cloudflare R2 | Resume storage, portfolio assets, generated PDFs |
| **PDF Generation** | Puppeteer (headless Chrome) | High-fidelity resume PDF output matching visual templates exactly |
| **Email** | Resend or SendGrid | Transactional emails (digest, milestone, alerts) with high deliverability |
| **Analytics** | Mixpanel or PostHog (self-hosted) | Granular user behavior tracking for feature optimization |
| **Deployment** | Vercel (frontend) + Railway / AWS (backend) | Vercel for instant frontend deployment; Railway for backend scaling |
| **Mobile** | React Native (Phase 2) or Progressive Web App (Phase 1) | PWA first for fast launch; native app for Phase 2 mobile push notifications |
